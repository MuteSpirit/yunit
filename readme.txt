yUnit (Yet another xUnit)

Features

Has common test runner for all unit engines (written in Lua).

Has own unit engine (for C++ and Lua):
    - short-spoken syntax
    - fixtures
    - marking test as ignored (it even will be not compiled)
    - marking test as TODO (you will get special message about it)
    - many assert functions 

Test fails messages in formats of SciTE and Visual Studio (jump to place by string double click)
