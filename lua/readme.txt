Описание внесенных изменений по сравнению с оригинальной версией Lua 5.1.4

1. Файл ldo.c: замена функции CRT exit() на __debugbreak()
2. Файл luaconf.h: макросы, отвечающие за совместимость с версией Lua 5.0
3. Файл luaconf.h: преобразование double->int выполняется компилятором (lua_number2int)


За совместимость отвечают макросы в luaconf.h:
    - совместимость по языку (функции стандартной библиотеки и конструкции языка):
        - LUA_COMPAT_GETN
        - LUA_COMPAT_LOADLIB
        - LUA_COMPAT_VARARG
        - LUA_COMPAT_MOD
        - LUA_COMPAT_LSTR
        - LUA_COMPAT_GFIND
    - совместимость по прикладному интерфейсу:
        - LUA_COMPAT_OPENLIB


На данный момент библиотека собрана без совместимости с 5.0 по языку (соответствующим макросам сделан undef в luaconf.h). Совместимость по прикладному интерфейсу оставлена, однако макрос LUA_COMPAT_OPENLIB убран из luaconf.h и теперь задается на уровне проекта.


Библиотека lua51.dll является прокси к lua5.1.dll - просто передает управление lua5.1.dll. Сначала необходимо собрать lua5.1.dll. Рекомендуется собирать lua51.dll только в release.
